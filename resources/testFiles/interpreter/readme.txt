Testing files for interpreter
each folder consists particular instructions:
heap - NEW, DEL, LDA, STA
IO - Read and write operations for int, real numbers and strings
intOperations - all OPRs
realOperations - all OPFs
conversions - ITR, RTI
loadStore - loading and storing int, real numbers
jumps - JMC, JMP
differentName - LDA, STA, PLD, PST
